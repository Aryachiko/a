Selamat datang! Untuk mengelola data pengunjung, Anda dapat menggunakan tombol-tombol yang tersedia di menu bar di sebelah kiri. Berikut adalah fungsi masing-masing tombol:

  • Home: Untuk kembali ke halaman selamat datang ini.
  • Tampil Pengunjung: Untuk melihat daftar semua data pengunjung yang tersimpan.
  • Input Pengunjung: Untuk menambahkan data pengunjung baru.
  • Edit Pengunjung: Untuk mengubah atau memperbarui data pengunjung yang sudah ada.
  • Delete Pengunjung: Untuk menghapus data pengunjung dari sistem.
